# Discord Auto-Line Bot
برمجة وتطوير: **Alexander & Founder Of Even**

## المميزات:
1. إرسال خط تلقائي في تصنيفات (Categories) محددة.
2. أمر `!say` لإرسال كلام متبوع بخط.
3. أمر `!addl` لإضافة رومات يدوية خارج التصنيفات.
4. أمر `!rl` لعرض الرومات المضافة بنظام نسخ سهل.
5. الرد التلقائي بالخط عند إرسال الإدارة لعلامة `-`.

## الأوامر:
- `!say "النص هنا" #channel`: إرسال رسالة لروم معين.
- `!addl`: إضافة الروم الحالي لقائمة الخط التلقائي.
- `!rl`: عرض جميع الرومات المضافة مع الـ ID.
